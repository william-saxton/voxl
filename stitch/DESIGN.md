# Design System Document

This document outlines the core aesthetic and functional principles of our design system, driven by the latest theme specifications.

## Theme Overview

Our design system is currently configured for a **dark** color mode, providing a sophisticated and comfortable viewing experience, especially in low-light environments.

## Color Palette

The color palette is built around a "fidelity" variant, originating from a seed color of `#7b4afd`.

-   **Primary Color:** `#7d4cff` (A vibrant, energetic purple, suitable for primary actions and branding accents.)
-   **Secondary Color:** `#7e6abe` (A muted, supporting purple-grey, designed for secondary actions and less prominent UI elements.)
-   **Tertiary Color:** `#b25700` (A warm, inviting orange, used for highlights, badges, and additional accentuation.)
-   **Neutral Color:** `#7a7582` (A balanced, cool grey, providing a stable foundation for backgrounds, surfaces, and typography.)

## Typography

Our typography system is built for clarity and modern appeal.

-   **Headlines:** `Inter` (A versatile and highly readable sans-serif, chosen for its modern feel and strong presence.)
-   **Body Text:** `Inter` (Ensuring consistency and legibility for all paragraph and long-form content.)
-   **Labels:** `Inter` (Providing clear and concise labels for interactive elements and data points.)

## Shape and Form

The system employs a **moderate (2)** level of roundedness for UI elements, striking a balance between modern aesthetics and approachability. This ensures elements have a subtle softness without appearing overly rounded or sharp.

## Spacing

Spacing within the UI is set to a **normal (2)** level, providing adequate whitespace for comfortable readability and visual separation between elements. This promotes a clean layout without feeling too dense or too sparse.
